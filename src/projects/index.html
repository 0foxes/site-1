Projects I've worked on in the past:

After first doing a test run, I created my own ethereum token on the Ropsten testnet. I started by setting up a wallet, then manually mining ETH for the contract gas. Then, I coded the smart contract in Solidity based off a template and deployed the contract on MyEtherWallet. Finally, I verified and published the source code, as is standard practice for many ethereum-based apps. The token may not have been very useful besides turning ETH into tokens, but this could be the foundation of something bigger, like a decentralised application (otherwise known as a DApp) or as a foundation for an ICO, which I later made.


Source Code: https://ropsten.etherscan.io/address/0x2eca6f051c395f3c94c86369dcf39765a5dabc42#code

Using Python, the module "requests", the Reddit API and matplotlib, I was able to code two programs to get and graph the number of active users on a subreddit, in real-time. One program made a request to Reddit, parsed the JSON response, then logged the time and the number of active users into a text file. The second program used matplotlib to plot and show a live graph of the number of online users. Although the two programs initially required Python to run, I was able to pack them into .exe's without dependencies with the help of PyInstaller.


Source Code: https://github.com//public-projects/tree/master/online-users

Machine learning and deep learning may be hyped, but the reality is they need a lot of computational resources (such as powerful GPUs) to run properly. My GPU was on the slow side and I had always wanted to try my hand at AI, so imagine my excitement when I learned about Google Colab, which lets regular people play with Tesla GPUs in the cloud through Jupyter Python notebooks. I also found GPT-2-Simple, a "simple" Python wrapper for GPT-2 (a popular deep learning language model). I started by downloading all the chat logs of a major Discord server in .txt format using this, which I then processed into "tokens" of training data. Then, I set up everything in the cloud, and started training it. Since training a model takes a while, it's still training as I type this. Hopefully it finishes before Google boots me for using too many resources...
