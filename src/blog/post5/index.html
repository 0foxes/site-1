Line by Line: Creating Your Own Website From Scratch (Part 1)

24/04/2020

Nowadays, everyone should have an internet presence. While some people and businesses settle for some social media pages, it’s better to have your own website. Your own website helps potential employers find you, showcases your skills to them (especially if you’re in the tech industry), and offers a place for you to share your thoughts with the world. So how can you build your own website?

While the numerous website builder services out there may look tempting (think Wix, Canva, or Wordpress), learning to build your own website from scratch is often more effective and rewarding. It’s truly your own personal website, you learn some coding while doing it, and it’s more customizable because you’re not picking from an array of pre-designed themes. This two-part guide helps you build your own website from scratch, even if you have zero technical knowledge. Part one (this) helps you get your website up and running, and part two helps you improve it.

Current students can sign up for the Github student pack, which lets you build your website for completely free. This guide assumes you have the Github student pack. If you don’t all the steps would still be the same, you’d just have to pay for it. To start, let’s familiarize ourselves with what any website needs. There are three main components: a domain, hosting, and content.

Domains: Just as your house has an address, your website needs an address on the internet. The address is called a domain. For example, my own website’s domain would be .com. Your domain name should ideally represent who you are or what you do, and you “register” a domain for increments of a year from domain registrars to use it. Domain registrars include NameCheap, Google Domains, and GoDaddy.

Hosting: Hosting is the server your website “lives” on 24/7 so everyone can access it. I recommend buying from DigitalOcean as you can get a lot of free credit there regardless of if you’re a student or not. If you’re a student you can get $50 that doesn’t expire (with the Github student pack, of course), and if you’re not you can still get $100 that expires in 3 months with my referral link. For reference, each month of hosting costs around $5.

Content: The content is what actually appears on the website. There are a lot of different programming languages that can make up a website, but you only need to know two of them: HTML and CSS. HTML controls the main content of your website (like the text, images, and links), and CSS controls the style, which is how they look (like the font size and colour).

Now that we know what a website needs, let’s build our own!

First, let’s get some hosting. Either use the Github student pack (recommended) to sign up for DigitalOcean or my referral link mentioned previously. Once we’re in, click the “Create Project” project button and call it whatever you want. After that, click “Get Started with a Droplet” to create your server. Recommended settings are as follows:

Choose an image --> Distributions: Ubuntu 18.04.3 (LTS) x64
Choose a plan --> Standard: $5/mo
Add block storage: None
Choose a datacenter region: The one closest to your geographical location
Select additional options: None
Authentication: One-time password

After it finishes creating the droplet, it’ll take you to a screen like this.

Click on your droplet name. After that, click “Console” to access your server. The username is “root”, and the password would have been emailed to you. Type the following commands:

“sudo passwd root” (to change your own password)
“sudo apt update” (to update the included software)
“sudo apt upgrade” (also to update the included software)
“sudo apt install apache2” (to install server software)
“curl -4 icanhazip.com” (to get your IP)

Note down your IP and type it in your browser’s address bar. You should be greeted with this page:

We now have hosting, but we need to connect it to a domain. If you have the Github student pack, go here and get your unique link for NameCheap. These instructions are specifically for NameCheap, but other domain registrars should have a very similar process. After you’ve registered a domain and paid, check your email to verify it. Then, go to your Dashboard and click “manage” next to your website name. Select “NameCheap Basic DNS” for your Nameservers.

Then follow these instructions to add a type "A" Record with your domain name and your DigitalOcean IP address. Apologies for linking an external guide, but I use an alternative DNS setup. Then, wait 5-10 minutes for your changes to take effect. When you go to your domain name now in your browser, you should see the Apache default page. Let’s switch that with something else.

You need to install WinSCP to update the files on your website. After it’s installed, click “New Site” and use the following settings:

Click “login”, and you should be able to see two windows of files. The files on the left are on your computer, and the files on the right are on the server. Go to /var/www/html/ on your server, and delete everything in that (that’s the default page). Finally, upload a file named “index.html” (you can learn HTML here, but plain text works as well. You now have your own website!

This was a relatively basic guide intended for people with little to no programming experience to make your own website, and I hope it worked well for you.
